#Basic NodeMCU Projects

This is the project containing some basic Arduino Projects done on NodeMCU during a 3-day IoT Workshop conducted in FISAT by ICT Academy of Kerala.

##Contents

*Projects using ThingSpeak
*Projects using Ubidot

##Prerequisites

*ArduinoIDE
*NodeMCU
*Install NodeMCU package from : http://arduino.esp8266.com/stable/package_esp8266com_index.json
*Basic sensors (HC-SR04, LDR, PIR Motion Sensor)
*Connecting Wires, Switches, Breadboard
*Basic knowledge on how to build circuits using breadboards
